# Roadmap

## Fase 1 completada en esta base

- arquitectura desacoplada;
- dos webparts;
- validaciones funcionales;
- modo mock rico;
- base SharePoint + Graph;
- documentacion de despliegue y modelo.

## Fase 2 recomendada

- People Picker real para asistentes;
- sincronizacion bidireccional con calendario;
- servicio de recomendaciones mas avanzado;
- filtros guardados por usuario en SharePoint o perfil;
- reporting de uso de salas;
- vista de incidencias con workflow.

## Fase 3 recomendada

- Azure Function para provisionamiento robusto;
- attendance real de reuniones de Teams;
- telemetria de negocio y Application Insights;
- integracion con reservas corporativas de espacios;
- automatismos de recordatorio, seguimiento y cierre.
