# Guia de administracion

## Tareas habituales

### Catalogos

- alta/baja de ubicaciones
- mantenimiento de salas
- mantenimiento de recursos y asignaciones
- activacion/desactivacion de plantillas

### Configuracion

- revision de `MM_MeetingConfigurations`
- gestion de filtros guardados base
- actualizacion de parametros por entorno

### Seguridad

- revisar grupos asignados
- auditar ruptura de herencia
- revisar accesos a incidencias y documentos sensibles

### Operacion

- revisar incidencias abiertas
- depurar reuniones con conflicto
- comprobar documentacion final pendiente

## Buenas practicas

- no editar a mano la estructura si existe script para ello;
- mantener codigos funcionales estables;
- no borrar maestros con datos dependientes sin analizar impacto.
