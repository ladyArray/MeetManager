# Checklist post-despliegue

## Estructura

- listas creadas correctamente
- bibliotecas creadas correctamente
- vistas visibles
- campos lookup resueltos
- content types documentales disponibles

## Seguridad

- grupos de SharePoint verificados
- herencia rota solo donde aplica
- acceso de usuarios validado

## Datos

- ubicaciones cargadas
- salas cargadas
- recursos cargados
- plantillas cargadas
- configuraciones base cargadas

## SPFx

- paquete cargado en App Catalog
- webparts visibles en paginas modernas
- propiedades configurables revisadas

## Integraciones

- permisos Graph aprobados si aplica
- enlace Teams visible en reuniones virtuales/hibridas
- disponibilidad de calendario validada en el entorno objetivo
